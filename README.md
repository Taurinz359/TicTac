Игра Крестики Нолики
===

Игра простая, все действия происходят в терминале. Первым ходит 'X', вторым 'O'

Дополнительно
----------
Должен быть установлен php7.4. & Composer 2.1.9

Установка
-----------

    $ cd /path/to/directory
    $ composer install
